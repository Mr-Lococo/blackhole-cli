# Security Policy

## Supported Versions

Here you can see the versions that will still be supported with security and TWO(Three-way-overight) updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.0-b1   | :white_check_mark: |
| 0.1.0-b0   | :x:                |
| T-10.b-1   | :x: |
| T-10.b   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities using GitHub's private vulnerability reporting feature.

Do not open public issues for security problems.
